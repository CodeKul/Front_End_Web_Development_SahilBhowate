The flex items are arranged along the main axis using the justify-content property. Depending on the direction of the main axis ; the ouput of the justify-content property will change.

The flex items are arranged along the cross axis using the align-items property. Depending on the direction of the cross axis ; the output of the align-items property will change.

How to identify the directions of main-axis ? It depends on the value of the flex-direction property. By default the value of flex-direction property is row.

How to identify the direction of cross-axis ? The cross axis is always perpendicular to the main axis. So , you have to identify the direction of the main axis first.

What does the flex-direction property decide ? It decides the direction in which the flex items would be displayed.
