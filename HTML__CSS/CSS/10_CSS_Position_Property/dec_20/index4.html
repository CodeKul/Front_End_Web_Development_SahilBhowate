If the parent of the position:absolute element has position:static ; then the position:absolute element goes out of proportion of the parent element. And the top/bottom ; left/right property of the position:absolute element acts according to its next parent element. Here , the next parent element is body tag.

If the parent of the position:absolute element has position:relative/sticky/fixed/absolute [means any non-static property] ; then the position:absolute element would be in the proportion of the parent element. And the top/bottom ; left/right property of the position:absolute element acts according to its parent element.
